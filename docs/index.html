Quick Start

1

Installation

# Install from npm (Recommended)
npm install -g @henteko/kumiki

# Verify installation
kumiki --version
2

Initialize and Configure

# Initialize a new project (creates sample project.json)
kumiki init my-project.json

# Configure Gemini API key (required for AI features)
kumiki config set gemini.apiKey YOUR_API_KEY
3

Generate Video

# Validate configuration
kumiki validate my-project.json

# Generate video
kumiki generate my-project.json
